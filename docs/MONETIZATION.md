# Monetization Strategy

## Tiers

### Free (npm package)
- Self-hosted via stdio transport
- All 5 tools included
- User provides own Shopify API key
- Distribution: npm, LobeHub marketplace

### Pro ($19/mo)
- Hosted SSE endpoint (no local setup)
- Web dashboard for configuration
- Usage analytics
- Priority support
- Stripe billing integration

### Enterprise (Custom)
- Multi-store support
- Custom tool development
- SLA guarantees
- Dedicated support

## Distribution Channels

1. **npm** — Direct installs
2. **LobeHub Marketplace** — MCP discovery
3. **GitHub** — Open source visibility
4. **Product Hunt** — Launch day
5. **X/Twitter** — MCP community
