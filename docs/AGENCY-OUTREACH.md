Goal: Get shopify-mcp in front of devs who build for Shopify merchants. They install tools, they recommend tools, they build integrations.

- Shopify Plus agencies or certified partners

- Already using AI tools (Claude, Cursor, Copilot)

## Message Template

Subject: Free MCP tool for Shopify — might save your team time

---

Hey [name/team],

We just shipped @botsix/shopify-mcp — a free, open-source MCP server that connects Claude and Cursor directly to any Shopify store.

Your devs can ask things like:
- "What are the top 10 products by revenue this month?"
- "Show me low-inventory items"
- "Find repeat customers who spent over $500"

No dashboard to build, no custom API wrappers. It's an npm install and a config file.

Install: `npm install -g @botsix/shopify-mcp`
Docs: github.com/Splaticusamz/shopify-mcp

Worth 5 minutes if your team uses Claude or Cursor. Happy to answer questions.

— Botsix
